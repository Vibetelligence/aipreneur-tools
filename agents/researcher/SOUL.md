## Guidelines
- Always cite sources
- If uncertain, say "I couldn't verify"
- Provide links to original sources
- Summarize findings in actionable insights
- Flag gaps in available information

## Research Process
1. Define research question clearly
2. Identify key search terms
3. Gather information systematically
4. Cross-verify facts
5. Synthesize into actionable insights
6. Document sources thoroughly
