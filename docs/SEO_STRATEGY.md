## Keyword Research Methodology

### Research Process

#### Step 1: Seed Keyword Generation

Start with core topics related to AI tools for business:

| Category | Seed Keywords |
|----------|---------------|
| Tools | AI software, AI apps, artificial intelligence tools |
| Business | AI for business, small business AI, AI automation |
| Specific | AI writing tools, AI video editing, AI design tools |
| Commercial | best AI tools, AI tool reviews, AI tool comparisons |
| Long-tail | AI tools for freelancers, free AI tools for business |

#### Step 2: Keyword Expansion

Use tools to expand seed keywords:

| Tool | Purpose | Use Case |
|------|---------|----------|
| Ahrefs | Keyword explorer | Volume, difficulty, clicks |
| SEMrush | Keyword magic tool | Related keywords, questions |
| AnswerThePublic | Question research | Content topics, FAQs |
| Google Autosuggest | Search suggestions | Real-time popular queries |
| Also Asked | Question clustering | PAA questions |

#### Step 3: Intent Classification

Categorize keywords by search intent:

| Intent Type | Description | Content Approach |
|-------------|-------------|------------------|
| Informational | Learning about AI tools | How-to guides, explainers |
| Commercial Investigation | Comparing options | Comparisons, reviews |
| Transactional | Ready to purchase | Product pages with affiliate links |
| Navigational | Seeking specific tools | Individual tool reviews |

#### Step 4: Prioritization Matrix

Score keywords based on:

| Factor | Weight | Scoring Criteria |
|--------|--------|------------------|
| Search volume | 25% | 1-10 scale |
| Difficulty | 25% | 1-10 scale (lower is better) |
| Intent match | 25% | 1-10 scale |
| Revenue potential | 25% | 1-10 scale |

**Priority Formula:** (Volume × 0.25) + ((10-Difficulty) × 0.25) + (Intent × 0.25) + (Revenue × 0.25)

## On-Page SEO Checklist

### Pre-Publish Checklist

| Element | Requirement | Status |
|---------|------------|--------|
| **Title Tag** | 50-60 characters, primary keyword near start | ☐ |
| **Meta Description** | 150-160 characters, compelling with CTA | ☐ |
| **URL Structure** | Clean, hyphenated, includes keyword | ☐ |
| **H1 Heading** | Single H1, includes primary keyword | ☐ |
| **H2/H3 Structure** | Logical hierarchy, keyword-rich headings | ☐ |
| **Keyword Usage** | Primary keyword 2-3 times, LSI keywords 5+ | ☐ |
| **Content Length** | 1,500+ words for reviews/guides | ☐ |
| **Internal Links** | 3-5 relevant internal links | ☐ |
| **External Links** | 2-3 authoritative external links | ☐ |
| **Images** | 3-5 images with descriptive alt text | ☐ |
| **Readability** | Grade 6-8 reading level | ☐ |
| **Mobile Optimization** | Responsive design, readable on mobile | ☐ |

### Title Tag Best Practices

| Element | Example |
|---------|---------|
| Format | Primary Keyword | Secondary Element |
| Review example | Jasper AI Review 2026: Is It Worth It for Business? |
| Comparison example | Jasper vs Copy.ai: Complete Comparison 2026 |
| Guide example | How to Use AI for Content Creation: Complete Guide |

### Meta Description Template

| Type | Template |
|------|----------|
| Review | [Tool Name] Review: [Key benefit]. [Number] features, pricing, and our verdict. Read our honest review. |
| Comparison | [Tool A] vs [Tool B]: [Key differentiator]. See our complete comparison with pricing and features. |
| Guide | Learn how to [benefit] with [topic]. Step-by-step guide for [audience]. No experience needed. |

### Heading Structure

```
H1: Primary Keyword - Compelling Title

H2: Introduction (150-200 words)
   └─ Hook + preview + structure

H2: Section Title (keyword-rich)
   └─ H3: Subsection (if needed)
   └─ H3: Subsection (if needed)

H2: Section Title (keyword-rich)
   └─ H3: Subsection (if needed)

H2: FAQ Section (if applicable)

H2: Final Verdict / Conclusion
```

## Technical SEO Requirements

### Site Architecture

| Element | Requirement | Priority |
|---------|-------------|----------|
| HTTPS | Full site SSL encryption | Critical |
| Mobile-first | Mobile responsive design | Critical |
| Site speed | <3 second load time | High |
| Core Web Vitals | Meet "good" thresholds | High |
| XML sitemap | Submitted to Search Console | Critical |
| Robots.txt | Properly configured | Critical |
| Canonical tags | Prevent duplicate content | High |
| Structured data | FAQ, Review markup | Medium |

### Core Web Vitals Targets

| Metric | Target | Current |
|--------|--------|---------|
| LCP (Largest Contentful Paint) | <2.5 seconds | - |
| FID (First Input Delay) | <100ms | - |
| CLS (Cumulative Layout Shift) | <0.1 | - |

### Site Speed Optimization Checklist

| Task | Impact | Effort |
|------|--------|--------|
| Image optimization (WebP) | High | Medium |
| Lazy loading for images | High | Low |
| Minify CSS/JavaScript | Medium | Low |
| Browser caching | Medium | Low |
| CDN implementation | High | Medium |
| Remove render-blocking resources | Medium | Medium |

### Structured Data Implementation

| Content Type | Schema Type | Priority |
|--------------|-------------|----------|
| Reviews | Review markup | High |
| Comparisons | Product/Comparison markup | High |
| How-to guides | HowTo schema | Medium |
| Articles | Article/BlogPosting | Medium |
| FAQs | FAQPage schema | Medium |

### Mobile Optimization

| Element | Requirement |
|---------|-------------|
| Responsive design | CSS media queries |
| Touch targets | 48x48px minimum |
| Font size | 16px minimum body text |
| Viewport meta tag | Required |
| No intrusive interstitials | Google policy compliance |

### Content Refresh Triggers

| Signal | Action |
|--------|--------|
| Traffic drop >20% | Update content, add new sections |
| Ranking drop >5 positions | Refresh keyword focus |
| New competitor ranking | Analyze gap, improve content |
| Outdated information | Update facts, add recent data |
| 6+ months since update | Scheduled refresh |

### Content Gap Analysis Process

1. **Identify:** Use Ahrefs/SEMrush to find ranking opportunities
2. **Analyze:** Review competitor content for gaps
3. **Prioritize:** Score by traffic potential and difficulty
4. **Create:** Produce content to fill gaps
5. **Optimize:** Internal link to new content
