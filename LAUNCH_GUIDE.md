## Step 3: Configure Domain

1. **Buy domain** from your preferred registrar
2. **Update DNS** to point to your hosting provider
3. **Enable HTTPS** (usually automatic)
4. **Set up www → non-www redirect**

**Cloudflare DNS example:**
| Type | Name | Target |
|------|------|--------|
| CNAME | @ | your-project.pages.dev |
| CNAME | www | your-project.pages.dev |

## Step 6: Submit to Google

1. Create Google Search Console account
2. Add your property (domain)
3. Verify ownership (DNS or file upload)
4. Submit sitemap: `https://yoursite.com/sitemap.xml`

**Sitemap URL:** You'll need to create one. For static HTML sites, use a simple XML sitemap generator.
